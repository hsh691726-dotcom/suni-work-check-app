# 총무회계 업무일지

총무회계 업무를 달력 중심으로 기록하고 확인하는 Streamlit 웹앱입니다.

처음 화면에서 바로 달력을 보고, 날짜를 클릭해서 그 날짜의 업무를 입력합니다. 한 날짜에 여러 업무를 기록할 수 있고, 매월 같은 날짜에 반복되는 업무는 한 번만 등록하면 매월 달력에 자동 표시됩니다.

## 주요 기능

- 첫 화면 달력 중심 구성
- 오늘 할 일, 이번주 할 일, 이달 할 일 대시보드
- 날짜 클릭 후 바로 업무 입력
- 한 날짜에 여러 업무 입력
- 업무유형, 진행상태, 중요도, 금액, 거래처, 메모 기록
- 매월 같은 날짜 반복 업무 자동 표시
- 선택 날짜 업무 목록 확인
- 상태 변경 및 삭제
- Google Sheets 저장/불러오기
- CSV 다운로드

## 데이터 저장 위치

앱 코드는 GitHub에 저장됩니다.

업무일지 데이터는 Google Sheets의 `journal_entries` 시트에 저장됩니다.

기존 실습에서 사용하던 `tasks` 시트는 건드리지 않고, 새 업무일지용 시트를 따로 사용합니다.

## 사용 방법

1. 왼쪽 저장 영역에서 Google Sheets 연결 상태를 확인합니다.
2. 달력에서 업무를 입력할 날짜를 클릭합니다.
3. 오른쪽 입력 영역에서 업무명을 입력합니다.
4. 업무유형, 진행상태, 중요도, 금액, 거래처, 메모를 입력합니다.
5. 매월 같은 날짜에 반복할 업무라면 `매월 같은 날짜 반복`을 `예`로 선택합니다.
6. `이 날짜에 업무 추가` 버튼을 누릅니다.
7. 다른 기기에서도 보이게 하려면 왼쪽의 `현재 업무일지 저장` 버튼을 누릅니다.
8. 다른 기기에서 최신 데이터를 보려면 `Google Sheets에서 불러오기`를 누릅니다.

## 반복 업무

예를 들어 매월 10일 원천세 확인 업무를 등록할 때:

- 날짜: 이번 달 10일
- 업무명: 원천세 신고/납부 확인
- 업무유형: 세무
- 매월 같은 날짜 반복: 예

이렇게 한 번만 등록하면 다음 달, 다다음 달 달력에도 자동 표시됩니다.

## 주의사항

- `.streamlit/secrets.toml`은 비밀키 파일이므로 GitHub에 올리면 안 됩니다.
- Google Sheets에 저장하려면 Streamlit Cloud의 Secrets 설정이 필요합니다.
- 앱에서 수정한 내용은 `현재 업무일지 저장`을 눌러야 Google Sheets에 반영됩니다.
